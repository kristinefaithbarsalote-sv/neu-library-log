📚 NEU Library Visitor Log System
A modern, web-based visitor logging system designed specifically for the New Era University Library. This application streamlines the check-in process for students, faculty, and staff while providing administrators with powerful data analytics.

🔗 Live Demo: https://im-project-7b2a1.web.app

✨ Key Features
Secure Authentication: Exclusive access for users with @neu.edu.ph email addresses via Google OAuth.
Role-Based Access: Separate interfaces for Visitors (logging entries) and Administrators (data analytics).
Real-Time Dashboard: Administrators can monitor live entry statistics, including total visits and breakdown by role (Student, Teacher, Staff).
Advanced Filtering: Filter logs by specific dates, colleges (BSIT, BSCS, etc.), or reasons for visiting (Research, Reading, etc.).
Export to PDF: Generate professional visitor reports with a single click using jspdf-autotable.

🛠️ Tech Stack
Frontend: React.js
Backend/Database: Firebase Firestore
Authentication: Firebase Auth (Google Provider)
Hosting: Firebase Hosting
Styling: Custom CSS (Responsive Design)

🚀 How to Run Locally
Clone the repository: git clone [your-github-link]
Install dependencies: npm install
Start the development server: npm start
